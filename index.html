HTML Shenanigans

  • <sub> for Subtext
  • <sup> for Supertext
  • <ins> for Underline
  • <center> to center text horizontally
  • <font color="red"> for colored text (red) (techn. deprecated, <p style="color:red;"> for text effects)
  • &nbsp HMTL escaped entities and symbols (i.e.non-breaking space to force spaces into text for indents)2
  • <img> to control image size etc.

HTML Tricks

  • <kbd> to symbolize keypresses

  • Dropdowns:

    <details>
<summary>Click Here to Expand!</summary>
Press <kbd>&darr;</kbd> to dive.
![](assets/Toicon.svg)[^subSource]
</details>

Math

  • $...$ or $`...`$ for inline math
  • $$ ... $$ or ```math ... ``` for blocks
  • Inline math like (2)(\frac{\pi}{2}) is inline
  • Blocks are centered: a2+b2=c2 a^2+b^2=c^2

Stuff for Issues

  • Task Lists with - [ ] ...
    • [x] for checked and [ ] for unchecked tasks
    • (GitLab) [~] for canceled tasks (strikethru)
  • References to issues with #<issueNr>
    • (GitLab) add + or +s for additional infos
  • Use keywords to close issues
    • e.g.fix, close, resolve
  • Custom diffs ```diff
    • (GitLab) create inline diffs with {- text -}

Special Quotes

  • Emotes (e.g.gitmoji.dev)
  • (GitHub) flavoured Quotes ([!NOTE], [!TIP], [!IMPORTANT], [!WARNING], [!CAUTION])
    • Alternative: custom quotes using emotes
      > :bulb: **Tip**
